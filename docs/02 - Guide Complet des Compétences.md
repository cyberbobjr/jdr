# Guide Complet des Compétences HARP Terre du Milieu
## 1. Règles Générales des Compétences

### Principes de Base
- Le joueur dispose de 84 Points de Développement (PdD) pour acquérir des compétences et des talents
- Les compétences gèrent les actions complexes nécessitant apprentissage et pratique
- Utilisées pour les manœuvres ayant une chance d'échec
- Chaque compétence est liée à deux caractéristiques
- Score final = somme des bonus (rangs + caractéristiques + autres)

### Calcul du Bonus Total de Compétence (BTC)
- BTC = Bonus de Rangs + Bonus de Caractéristiques + Bonus Spéciaux
- Maximum 6 rangs au niveau 1 (incluant rangs d'adolescence)
- Les 10 premiers rangs donnent +5 chacun
- Coût en Points de Développement :
  - Groupe favori : 2 Points de Développement (PdP) par rang
  - Hors groupe : 4 Points de Développement (PdP) par rang

### Seuil de Réussite
- En général, un score de 100+ est requis pour réussir
- Modificateurs selon situation et difficulté

### Types de Résolution
| Type | Description | Usage |
|------|-------------|-------|
| Tout-ou-rien | Simple réussite/échec | Actions générales |
| Combat | Tables spécifiques | Actions de combat |
| Pourcentage | Degré de réussite variable | Evaluation, négociation |
| JR | Jet opposé | Actions contre résistance |
| Spécial | Règles uniques | Cas particuliers |
| Utilitaire | Pour sorts mineurs | Magie simple |

### Difficultés Standard
| Niveau | Modificateur |
|--------|--------------|
| Routine | +60 |
| Très Facile | +40 |
| Facile | +20 |
| Moyenne | +0 |
| Difficile | -20 |
| Très Difficile | -40 |
| Extrêmement Difficile | -60 |
| Pure Folie | -80 |
| Absurde | -100 |

## 2. Notes Importantes
- Les compétences marquées () requièrent une spécialisation
- Les bonus de caractéristiques s'additionnent
- Les sous-compétences utilisent le BTC parent avec malus indiqué
- Les compétences de groupe favori coûtent moitié prix
- Maximum 6 rangs au niveau 1 inclut tous les bonus
- Les premiers rangs (1-10) donnent toujours +5 chacun
- Certaines compétences ont des règles spéciales uniques
